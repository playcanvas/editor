Version Control Graph - Node And Menu Proposal

Same graph geometry and menu anchoring; only the node/menu theme is modernized.

Host Panel

`vc-graph-panel` is fixed at 95% width/height, centered by translate(-50%, -50%). The graph fills it; the close button is appended into the graph DOM.

Node Contract

Nodes are JointJS SVG `html.Element` groups. The graph renders SVG rect/text plus a `foreignObject` containing `.graph-node-div`, which can host richer picker-style HTML.

Layout Math

Graph coords become pixels with x = 265 * coord + 80 and y = 110 * coord + 50. Current node size is 190x72; the proposal uses 226x86.

Edges And Menu

Edges use branch color, stroke width 3, markers, and JointJS anchors. Node clicks show a PCUI menu; richer node content should stay inside the graph node's `foreignObject`.

Modernized proposal

Dark SVG shell with branch accents; picker-style HTML content inside the node foreignObject.
